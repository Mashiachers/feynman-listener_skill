# Feynman Listener examples

Use these examples to calibrate specificity. The key pattern is: quote evidence, ask one focused question, then explain directly after the learner answers.

## 1. Math / optimization: gradient descent

Learner's first explanation:

> 梯度下降就是看当前位置斜率往哪里上升最快，然后往反方向走一点。这样函数值会下降，重复很多次就会靠近最小值。学习率太大时会跳过最低点。

Good AI probe response:

```markdown
我听到的理解：你已经抓住了“负梯度方向让函数下降”的直觉，但还没有区分“每步下降”“收敛到最小值”和“只到局部低点”。

做得好的地方：
- 你把梯度解释成“上升最快方向”，不是只背公式。
- 你提到了学习率过大导致跳过最低点，这是一个重要边界。

我想追问的地方：
1. [机制] 证据：“这样函数值会下降，重复很多次就会靠近最小值”
   问题：对一维函数 f(x)=x^2，如果 x=3、学习率 α=0.1，下一步 x 会变成多少？这个数字怎样说明“负梯度方向让函数值下降”？

2. [边界] 证据：“靠近最小值”
   问题：如果函数不是凸的，梯度下降为什么可能只找到局部最小值，而不是全局最小值？

3. [准确性] 证据：“学习率太大时会跳过最低点”
   问题：还是用 f(x)=x^2，为什么 α=1.1 会让 x 在正负之间越跳越大？
```

After the learner answers:

```markdown
判断：基本正确

直接答案：对 f(x)=x^2，梯度是 2x。若 x=3 且 α=0.1，则 x_new = 3 - 0.1*6 = 2.4，函数值从 9 降到 5.76。这说明沿负梯度走一小步，确实能降低函数值。

为什么：梯度告诉你“往哪边上升最快”，所以负梯度就是局部最陡下降方向。但这只是局部判断，不保证跨过很远距离后仍然下降。

需要修正/补强：你说“会靠近最小值”时要加条件：步长合适、函数形状较好，例如凸函数；非凸时可能卡在局部最小或鞍点附近。

下一轮可以这样讲：梯度下降每次用当前位置的斜率做一个局部下降决策，例如 f(x)=x^2 从 3 走到 2.4，函数值变小；但它能否收敛取决于步长和函数形状。

下一步：把“最小值”改成“一个局部低点或在凸条件下的全局最小值”。
```

## 2. Engineering / signals: distortionless transmission

Learner's first explanation:

> 无失真传输就是信号通过系统以后形状不变，只是幅度和时间可能变了。系统的频率响应应该所有频率都一样。

Good AI probe response:

```markdown
我听到的理解：你知道无失真传输允许幅度缩放和延时，但“所有频率都一样”这句话还需要拆成幅频和相频两个条件。

做得好的地方：
- 你没有把“无失真”误解成输出必须和输入完全相等。
- 你提到了“时间可能变了”，这暗示你知道延时是允许的。

我想追问的地方：
1. [定义] 证据：“只是幅度和时间可能变了”
   问题：能不能把无失真传输写成 y(t)=Kx(t-t0)？这里 K 和 t0 分别代表什么？

2. [机制] 证据：“频率响应应该所有频率都一样”
   问题：为什么幅度响应应该是常数 |H(ω)|=K，但相位响应不是常数，而应该是 -ωt0 这样的线性相位？

3. [边界] 证据：“所有频率都一样”
   问题：如果系统对高频和低频的幅度增益一样，但相位不是线性的，输出波形为什么仍然会失真？
```

After the learner answers:

```markdown
判断：部分正确

直接答案：无失真传输的核心是 y(t)=Kx(t-t0)。在频域中，这对应 H(ω)=Ke^{-jωt0}，所以幅度响应是常数 K，相位响应是线性函数 -ωt0。

为什么：不同频率分量共同组成原波形。幅度常数保证各频率比例不变；线性相位保证所有频率分量具有同一个时间延迟 t0。如果相位乱了，不同频率到达时间不同，叠加出的波形就变形。

需要修正/补强：“所有频率都一样”只适合描述幅度，不适合描述相位。相位要线性，不是相同。

下一轮可以这样讲：无失真不是输出完全不变，而是只允许整体放大 K 和整体延时 t0；因此频率响应必须有恒定幅度和线性相位。

下一步：用一个由两个正弦波相加的信号，想象其中一个频率被额外延迟时波形会怎样改变。
```

## 3. Physics: Gauss's law and electric flux

Learner's first explanation:

> 高斯定律就是穿过封闭曲面的电场线数量等于里面的电荷。电荷越多，电通量越大。球面对点电荷很好算，因为到处一样。

Good AI probe response:

```markdown
我听到的理解：你抓住了“封闭曲面”和“包围电荷”这两个关键词，但还需要区分电场强度 E 与总通量 Φ_E。

做得好的地方：
- 你知道高斯面必须是封闭曲面。
- 你能说出球面对点电荷适合使用对称性。

我想追问的地方：
1. [定义] 证据：“电场线数量等于里面的电荷”
   问题：如果用公式写，高斯定律为什么是 ∮E·dA = Q_enclosed/ε0，而不是直接说 E = Q/ε0？

2. [机制] 证据：“球面对点电荷很好算，因为到处一样”
   问题：球面上“到处一样”的到底是 E 的大小、方向，还是 E·dA？这一步为什么能把 E 从积分号里拿出来？

3. [边界] 证据：“电荷越多，电通量越大”
   问题：如果封闭曲面外面放一个很大的电荷，它会不会改变通过这个曲面的净通量？为什么？
```

After the learner answers:

```markdown
判断：部分正确

直接答案：外部电荷会改变曲面上各点的电场，但不会改变净通量，因为它的电场线进入封闭曲面多少，就会离开多少，净贡献为零。

为什么：高斯定律关心的是封闭曲面的总通量，只由内部净电荷决定。外部电荷可以让局部 E 变大或变小，但正负通量会抵消。

需要修正/补强：不要把“电场强度”和“总通量”混为一谈。E 是局部量，通量积分是整体量。

下一轮可以这样讲：高斯定律说封闭曲面的净电通量只由内部净电荷决定；外部电荷会改变曲面上的局部电场分布，但不会改变总净通量。

下一步：画一个外部正电荷的电场线，标出哪些线进入高斯面、哪些线离开。
```

## 4. Computer science: database transaction isolation

Learner's first explanation:

> 事务隔离就是多个事务同时执行的时候不要互相影响。Serializable 最严格，相当于一个一个执行。Read committed 就是只能读到已经提交的数据。

Good AI probe response:

```markdown
我听到的理解：你能按强弱区分隔离级别，但还没有用具体并发异常说明它们分别防住什么、防不住什么。

做得好的地方：
- 你准确提到了 Read Committed 的“只能读已提交数据”。
- 你知道 Serializable 要表现得像某种串行执行。

我想追问的地方：
1. [例子] 证据：“不要互相影响”
   问题：能不能用两个事务同时修改账户余额，举一个 lost update 的例子？

2. [边界] 证据：“Read committed 就是只能读到已经提交的数据”
   问题：Read Committed 能防 dirty read，但为什么仍可能出现 non-repeatable read？

3. [机制] 证据：“Serializable 最严格，相当于一个一个执行”
   问题：“相当于一个一个执行”是不是等于数据库真的不并发执行？它和冲突检测、锁或 MVCC 的关系是什么？
```

After the learner answers:

```markdown
判断：基本正确

直接答案：Read Committed 防止读到未提交数据，但同一个事务内两次读取同一行时，中间如果另一个事务提交了修改，第二次读可能得到新值，所以会出现 non-repeatable read。

为什么：Read Committed 的承诺是“每次读都读当时已经提交的数据”，不是“整个事务期间看到同一个快照”。

需要修正/补强：把隔离级别讲清楚时，要用异常来定位：dirty read、non-repeatable read、phantom read、lost update，而不是只说“互相影响”。

下一轮可以这样讲：Read Committed 像是每次 SELECT 前都重新看一眼已提交世界，所以不会读脏数据，但两次 SELECT 之间世界可能已经变了。

下一步：把四种并发异常各配一个两事务时间线。
```

## 5. Biology / biochemistry: Michaelis-Menten kinetics

Learner's first explanation:

> 米氏方程描述酶反应速度和底物浓度的关系。底物越多速度越快，但最后会饱和，因为酶都被占满了。Km 是反应速度到最大速度一半的时候的底物浓度。

Good AI probe response:

```markdown
我听到的理解：你能说出饱和曲线和 Km 的操作性定义，但还没有解释 Km 与亲和力的关系为什么需要小心。

做得好的地方：
- 你把“底物越多越快”补充成了“最后会饱和”，没有停留在线性直觉。
- 你知道 Km 对应 v=Vmax/2 时的底物浓度。

我想追问的地方：
1. [机制] 证据：“因为酶都被占满了”
   问题：当酶接近饱和时，为什么继续增加底物浓度不能让速度无限增加？

2. [定义] 证据：“Km 是反应速度到最大速度一半的时候的底物浓度”
   问题：请把这句话和公式 v=Vmax[S]/(Km+[S]) 对应起来：当 [S]=Km 时，v 为什么等于 Vmax/2？

3. [边界] 证据：“Km”
   问题：为什么常说 Km 越小“表观亲和力”越高，但不能在所有机制下直接把 Km 等同于结合常数 Kd？
```

After the learner answers:

```markdown
判断：部分正确

直接答案：当 [S]=Km 时，v=Vmax[S]/(Km+[S]) 变成 v=Vmax*Km/(Km+Km)=Vmax/2。Km 小表示达到半最大速度所需底物浓度低，通常意味着酶在较低底物浓度下就很有效。

为什么：饱和来自酶活性位点数量有限。底物再多，也要等酶完成催化、释放产物后才能处理下一批。

需要修正/补强：Km 可以反映表观亲和力，但它还受到催化步骤影响；只有在某些简化条件下才接近 Kd。

下一轮可以这样讲：米氏方程描述的是有限酶位点造成的饱和关系；Km 是半最大速度对应的底物浓度，常用于比较表观亲和力，但不是无条件的结合常数。

下一步：画一条 v 对 [S] 的曲线，标出 Vmax 和 Vmax/2。
```

## 6. History: Treaty of Versailles and World War II

Learner's first explanation:

> 一战后的凡尔赛条约惩罚德国太重，导致德国人不满，所以后来希特勒上台，最后爆发二战。

Good AI probe response:

```markdown
我听到的理解：你给出了“条约惩罚→德国不满→纳粹上台→二战”的主线，但目前因果链太单线，需要加入经济危机、政治制度和国际秩序的中间环节。

做得好的地方：
- 你没有把二战原因完全归结为某一个人的决定。
- 你抓住了战后安排与德国国内情绪之间的联系。

我想追问的地方：
1. [机制] 证据：“惩罚德国太重，导致德国人不满”
   问题：条约中的赔款、领土损失和战争罪责条款，分别怎样转化为德国国内的政治动员资源？

2. [边界] 证据：“所以后来希特勒上台”
   问题：如果只有凡尔赛条约，没有 1929 年经济大萧条，纳粹上台是否同样不可避免？请说明中间机制。

3. [准确性] 证据：“最后爆发二战”
   问题：从纳粹上台到二战爆发，中间还有哪些国际绥靖或集体安全失败的事件不能省略？
```

After the learner answers:

```markdown
判断：部分正确

直接答案：凡尔赛条约是重要背景，但不是单独充分原因。它制造了民族屈辱感和修约诉求；大萧条扩大失业和制度失信；魏玛政治碎片化让极端政党更容易动员；国际社会对扩张的软弱反应又降低了侵略成本。

为什么：历史因果通常不是一条直线，而是多个条件叠加后打开可能性空间。凡尔赛条约解释“为什么有怨恨”，但还不能单独解释“为什么纳粹此时成功夺权并发动战争”。

需要修正/补强：把“导致”改成“为……提供条件”，会更准确。

下一轮可以这样讲：凡尔赛条约提供了德国修约主义和民族主义动员的土壤，但纳粹上台还依赖大萧条、魏玛制度危机和保守精英误判，二战爆发还涉及国际秩序对侵略的失败约束。

下一步：按“战后安排—国内危机—政党动员—国际反应”四层重讲。
```

## 7. Humanities / literature: Lu Xun's "A Madman's Diary"

Learner's first explanation:

> 《狂人日记》是在批判封建礼教吃人。狂人看起来疯了，但其实他说出了社会真相。吃人就是压迫人的意思。

Good AI probe response:

```markdown
我听到的理解：你抓住了“礼教吃人”的核心隐喻，但还需要说明文本为什么要用“狂人”这个不可靠视角来表达真相。

做得好的地方：
- 你没有把“吃人”只理解成字面情节。
- 你注意到了“疯癫”和“真相”之间的张力。

我想追问的地方：
1. [机制] 证据：“狂人看起来疯了，但其实他说出了社会真相”
   问题：为什么鲁迅不让一个完全清醒的叙述者直接批判礼教，而要通过“狂人”的日记来讲？

2. [例子] 证据：“吃人就是压迫人的意思”
   问题：能不能从文本里举一个具体细节，说明“吃人”如何从幻觉变成对社会关系的隐喻？

3. [边界] 证据：“封建礼教吃人”
   问题：如果只说“封建礼教坏”，会漏掉这篇小说对旁观者、家族关系和日常伦理的哪一层批判？
```

After the learner answers:

```markdown
判断：基本正确

直接答案：“狂人”视角让文本同时拥有两层效果：表面上他的话可以被社会当成疯话排除，深层上读者却能看到疯话揭露了正常社会的残酷结构。

为什么：如果清醒叙述者直接说教，小说会变成议论文；狂人的日记让“看见真相的人被判为疯子”本身成为批判对象。

需要修正/补强：“吃人=压迫人”还太抽象，最好落到家族、邻里、历史书写和伦理话语怎样共同维持压迫。

下一轮可以这样讲：《狂人日记》不只是喊出礼教吃人，而是用一个被视为疯子的叙述者，展示正常社会如何把残酷包装成伦理和常识。

下一步：找一个文本细节，把“吃人”从口号讲成具体阅读证据。
```
